**************************************************
*** Installing and starting Vanted under Linux ***
**************************************************

type in shell:
$: unzip vanted.zip
$: chmod +x startVanted.sh
$: startVanted.sh

if you need more memory you have to edit "startVanted.sh"

****************************************************
*** Installing and starting Vanted under Windows ***
****************************************************

unzip vanted.zip by double-clicking
start startVanted.cmd by double-clicking

if you need more memory you have to edit "startVanted.cmd"
